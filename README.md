# KisanConnect

KisanConnect is a web application designed to connect farmers with buyers, allowing farmers to list their crops and buyers to view and contact them easily. This application aims to streamline the process of buying and selling agricultural products.

## Features

- **Farmer Registration**: Farmers can register and add their crops to the platform.
- **Crop Listings**: Buyers can view available crops along with details such as price, quantity, and location.
- **Contact Farmers**: Buyers can contact farmers directly through the application to inquire about crops.
- **User-Friendly Interface**: The application is designed with a simple and intuitive user interface for easy navigation.

## Technologies Used

- **Frontend**: React.js
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **Styling**: CSS

## Getting Started

### Prerequisites

- Node.js
- MongoDB

### Installation

1. Clone the repository:
   ```
   git clone <repository-url>
   cd KisanConnect
   ```

2. Install server dependencies:
   ```
   cd server
   npm install
   ```

3. Install client dependencies:
   ```
   cd client
   npm install
   ```

4. Set up environment variables:
   - Copy `.env.example` to `.env` in the server directory and configure your MongoDB connection string.

### Running the Application

1. Start the server:
   ```
   cd server
   npm start
   ```

2. Start the client:
   ```
   cd client
   npm start
   ```

The application will be running on `http://localhost:3000`.
